# 部活紹介

## 概要

- ViteのPreactで作成しました
- UIはtailwindcssを使っています
- 言語はTypescriptです
- bunを使用しています

## 使い方 / 開発の仕方

GitリポジトリをクローンしてNode.jsかBunを入れてください。

### 依存パッケージのインストール

Node.jsの場合

```
npm install
```

Bunの場合

```
bun install
```

### 開発サーバーの起動

Node.jsの場合

```
npm run dev
```

Bunの場合

```
bun run dev
```
